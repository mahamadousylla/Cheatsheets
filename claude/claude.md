# Claude

Ctrl + O : Showing detailed transcript

? : See all shortcuts

Shift + Tab : To cycle between: Default mode / Plan mode / Accept Edits (Code mode)

/clear : Clears the conversation history and free up context

/login : Login to your Anthropic account

/plugins : To view or add new marketplaces

/rewind : Restore the conversation/code to a previous point

/security-review : Complete a security review of the pending changes in the current branch. An agent is spun up for this

/stats : Show your Claude Code usage statistics and activity. Good if you want to see if you're near your limits

# Claude Skills

Claude Skills are Plugins for Claude that teach it new skills
